# 提示工程指南

提示工程是一門相對而言較新的學科，著眼於開發和最佳化提示，以在多樣化的應用和研究主題中更有效率地使用`語言模型 (LM)`。提示工程的技術，有助於更好地理解`大型語言模型 (LLM)`的能力和限制。

研究人員使用提示工程來提升 LLM 在各種常見和複雜的任務上的能力，像是問答和算式推理。開發人員使用提示工程來設計強健而有效的提示技術，實現和 LLM 及其他工具間的銜接。

提示工程不只是關於設計和開發提示，它涵蓋了與 LLM 互動和開發所需的廣泛技能和技術。這是一個對於介面與 LLM 銜接、建立和理解 LLM 能力來說都很重要的技能。你能使用提示工程來提升 LLM 的安全性和建構新功能，如：將 LLM 與領域知識和外部工具互相結合。

我們受到對 LLM 在開發上高度興趣的驅動，創建了這個新的提示工程指南，包含：與提示工程相關的最新論文、學習指南、模型、講座、參考資料、新的 LLM 功能及工具。
