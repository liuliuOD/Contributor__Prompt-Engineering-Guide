# 簡介

提示工程是一門相對而言較新的學科，著眼於開發和最佳化提示，以在多樣化的應用和研究主題中更有效率地使用`語言模型 (LM)`。提示工程的技術，有助於更好地理解`大型語言模型 (LLM)`的能力和限制。研究人員使用提示工程來提升 LLM 在各種常見和複雜的任務上的能力，像是問答和算式推理。開發人員使用提示工程來設計強健而有效的提示技術，實現和 LLM 及其他工具間的銜接。

這個指南涵蓋提示的基礎知識，提供如何使用提示來和 LLM 互動和指示的粗略想法。

全部的範例都是以 `text-davinci-003` 進行測試（使用 OpenAI 的遊樂場），除非另有說明。皆使用默認的參數設定，如：`temperature=0.7` 和 `top-p=1`。
